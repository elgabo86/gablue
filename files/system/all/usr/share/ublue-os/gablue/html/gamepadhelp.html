Mode d'Emploi : Raccourcis à la Manette

Raccourcis Principaux

Tous les raccourcis sont activés en maintenant le bouton Home pressé, combiné avec un autre bouton ou direction.

Home + Select Tuer tous les processus des jeux windows ou des émulateurs
Home + Start Ouvrir EmulationStation (ES)
Home + R3 Activer/Désactiver le mode souris (utiliser la manette comme souris)
Home + L3 Activer/Désactiver le mute audio
Home + △ Lancer ou fermer Youtube en plein écran
Home + ○ Ouvrir le menu pour désactiver les manettes bluetooth et redémarrer ou éteindre la machine
Home + ← Prendre une capture d'écran
Home + → Démarrer/Arrêter l'enregistrement vidéo
Home + ↑ Changer la limite FPS (60/120/sans limite)
Home + ↓ Afficher/Cacher MangoHud (overlay FPS) - Permet de basculer entre mode horizontal, vertical et caché
Home + Stick Gauche ↑ Augmenter le volume (+10%)
Home + Stick Gauche ↓ Diminuer le volume (-10%)

Mode Souris (Activé par Home + R3)
